# InkSight Legal & Support

Self-contained GitHub Pages site for InkSight's public Privacy Policy and Support URLs.

Live site: <https://sabaaziz991-hash.github.io/inksight-legal/>

## Publish

The site is published from the `main` branch of
[`sabaaziz991-hash/inksight-legal`](https://github.com/sabaaziz991-hash/inksight-legal).

Public metadata URLs:

- <https://sabaaziz991-hash.github.io/inksight-legal/privacy-policy/>
- <https://sabaaziz991-hash.github.io/inksight-legal/support/>

Push updates to `main`; GitHub Pages rebuilds the site automatically.

Do not publish the parent handwriting app workspace as part of this repository. It contains research/source material that is unrelated to the public legal site.
