# InkSight Legal & Support

Self-contained GitHub Pages site for InkSight's public Privacy Policy and Support URLs.

## Publish

1. Create a separate public GitHub repository, recommended name: `inksight-legal`.
2. Copy the contents of this directory to the repository root.
3. Commit and push the files to the repository's default branch.
4. In GitHub: **Settings → Pages → Build and deployment → Source: Deploy from a branch**.
5. Select the default branch and `/ (root)`, then save.
6. Verify these public URLs:
   - `https://GITHUB_USERNAME.github.io/inksight-legal/privacy-policy/`
   - `https://GITHUB_USERNAME.github.io/inksight-legal/support/`
7. Replace the `.invalid` metadata URLs in the iOS project and App Store Connect.

Do not publish the parent handwriting app workspace as part of this repository. It contains research/source material that is unrelated to the public legal site.
