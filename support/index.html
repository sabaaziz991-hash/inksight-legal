Support for InkSight on iPhone

How can we help?

Most issues can be resolved with the steps below. If you still need help, open a support issue and include the app version, iOS version, and a description of what happened.

Protect your privacy. Do not attach handwriting scans, exported reports, names, addresses, or other sensitive personal information to a public GitHub issue.

Scanning and photo recognition

  • Use even lighting and avoid shadows across the writing.
  • Keep all four page edges visible and hold the camera parallel to the page.
  • Use the border tool if automatic page detection selects the wrong area.
  • Try enhancement for faint writing; turn it off if dark paper or colored ink loses detail.
  • For best results, use several lines of handwriting rather than a single word.

Reports and local data

Reports are saved locally on the device. InkSight does not provide an account or cloud sync. Export a PDF or Markdown copy when you need to keep a report outside the app.

If a report cannot be saved, use the Retry action shown in the result screen. If an older report cannot be read after an update, InkSight keeps the remaining reports available and displays a warning.

Premium subscription

If an active subscription is not recognized, open Profile in InkSight and choose Restore Purchases. Confirm that the device uses the Apple Account that made the purchase.
